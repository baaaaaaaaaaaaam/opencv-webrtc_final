# 앱소개
이 앱은 opencv를 사용하여 얼굴을 검출한 후 검출한 얼굴 좌표의 비율을 찾아 원하는 이미지를 합성하는 앱 입니다. 추가로 AppRTC라이브러리를 사용하여 유저간 영상통화를 할 수 있고 영상통화도중에도 이미지 합성을 진행하엿습니다.

# 기능



#사용기술

● 서버: AWS (Linux)

● 웹 서버: apache,

● 데이터베이스: Mysql

● 언어: Java, php, 

● 라이브러리/API : AppRTC, opencv
